# 2. - Install 

Add this to your `repositories` in `composer.json`

```json
{
    "type": "composer",
    "url": "https://raw.githubusercontent.com/LoomZero/zero-composer-register/master/"
}
```

Execute:

`composer require loomzero/zero_ajax_api`

# 3. - Update

Execute:

`composer update "loomzero/zero_ajax_api"`
